### How to install?

Make sure you have Node.js on your device: [install node.js](https://nodejs.org/en/download)

After Git clone,
Go to project command line root
Execute :
```shell
yarn
```

### Run project in local 
Run:
```shell
yarn start
```
The project will automatically open as http://localhost:3000
